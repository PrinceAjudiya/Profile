Place Alarm Application

The purpose this application is travelling, when the user travel from one place to another place at that time this application is very helpful if user is going there at first time. At that time user can set alarm to remind them for their destination and also user can see the distance between destination from their current location. Also, using the feature of Google map user can easily find places and can see how far it is from their current location.
